Gemini API Key: 본인의 재미나이 AI의 API 키 값일 입력한다.

설명 이미지 업로드 : 본인지 참고 하여 작성할 기획서의 디스크립션을 기입한다. 다만 너무 길고 많은 정보의 값이 들어있는 이미지 첨부 시 작성된 값이 짤려있을 가능성이 높다. 분할 및 나누어 진행하는것이 정신적으로 안정

예시 엑셀/CSV 파일 업로드 : 본인이 작성한 엑셀 또는 CSV 파일을 첨부한다. 프롬프트에 기입된 내용과 같이 카테고리 순서를 맞춰서 작성된 파일을 추천한다.

프롬프트 : 본인이 만든 프롬프트를 입력한다.

AI 실행 : 키값을 입력한 AI가 프롬프트를 읽고 동작하여 우측 결과를 노출 시킨다. 시간은 대략 1~2분 내외
